## Projects Showcase

This section is for club project write-ups, demo pages, and build logs.

- Add a project page to `src/content/projects`
- Share what you built, why it matters, and what you learned
- Link to your repo or live demo when possible

## Start a project page

Create a Markdown file like `src/content/projects/project-name.mdx` and include:

- project title
- description
- status or links
- screenshots or code samples
