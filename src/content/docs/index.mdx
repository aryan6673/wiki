## Welcome

This wiki is built with Astro and Starlight so Hack Club members can add content using Markdown.

Use the sidebar to explore community guides, project write-ups, and club resources.

<CardGrid stagger>
	<Card title="Create new pages" icon="add-document">
		Add Markdown files under `src/content/community` or `src/content/projects`.
	</Card>
	<Card title="Organize your wiki" icon="folder">
		Update `astro.config.mjs` to add sections and navigation.
	</Card>
	<Card title="Share with the club" icon="chat-bubble">
		Use this site to publish club events, tutorials, and member projects.
	</Card>
	<Card title="Keep it open" icon="sparkles">

	</Card>
