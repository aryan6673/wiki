## How to Contribute

Welcome! If you have something to share with Hack Club, add it here.

### What to write

- Event recaps
- Tips for new Hack Club members
- Project ideas or workshop notes
- Club culture and community guides

### How to add a page

1. Create `src/content/community/your-topic.mdx`
2. Add frontmatter with a `title`
3. Write your content in Markdown
4. Save and preview locally with `npm run dev`

### Example frontmatter

```md
---
title: My First Hack Club Post
description: An intro to contributing to the wiki.
---
```
