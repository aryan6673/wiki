## Community Hub

This section is for club announcements, meetup notes, mentorship tips, and member-written guides.

- Add a new page by creating a Markdown file in `src/content/community`
- Keep entries short, friendly, and helpful for Hack Club members
- Use headings, links, and lists to make content easy to scan

## Get started

1. Create a new file like `src/content/community/your-topic.mdx`
2. Add a `title` frontmatter field
3. Write your content in plain Markdown
4. Commit and share it with the club!
